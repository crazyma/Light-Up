# Light-Up
a simple widget app to light up the screen of your android phone

## Why to build this app
主要是因為現在很多刷卡集點的時候，需要亮出barcode or BarCode<br/>
常常會被店員要求把螢幕調亮一點<br/>
覺得很麻煩，所以就做了一個 widget 省時間 (倒底是有多懶 XD)
